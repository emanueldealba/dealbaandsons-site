# Stripe Payment Links

Paste your real Stripe Payment Link URLs here and then replace the placeholder values in `payment-links.js`.

Recommended links to create in Stripe:

1. OVA Deposit / Fit Review Payment Link
2. Opportunity Viability Assessment Payment Link
3. Validation Sprint Payment Link, optional
4. Launch Readiness Blueprint Payment Link, optional

Do not place Stripe secret keys, API keys, or webhook secrets in this repo. Use only public Stripe Payment Link URLs that start with `https://buy.stripe.com/`.

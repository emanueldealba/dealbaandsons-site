# DeAlba & Sons Premium Static Site

## Deploy on Netlify
1. Unzip this folder.
2. Drag the `dealba_premium_site` folder into Netlify deploy.
3. Confirm the form appears under Netlify Forms after first submission.
4. Calendly founder launch link has been installed. Replace placeholder pricing, final email, and social links when available.

## Favicon verification

This build includes a full favicon set: `/favicon.ico`, `/favicon.svg`, PNG icons, `apple-touch-icon.png`, `site.webmanifest`, and `browserconfig.xml`. After deploying, open `https://dealbaandsons.com/favicon.ico` directly once, then hard-refresh the site. Browser tabs can cache old favicons aggressively for several hours.
